Test The Web Forward介绍

Test the Web Forward活动的意图是让关注Web标准的开发者直接参与到Web平台的贡献中去。W3C、Adobe、Intel、Mozilla 等的技术专家，现场帮助开发者解决活动中遇到的难题。在这次Hack活动中，你会了解最新的HTML5和CSS3功能规范，并学习如何编写对应的测试用例，这些测试用例会帮助更多浏览器更一致地实现这些功能。现场会有众多专家会帮助你完成HTML5和CSS3相关规范的测试用例编写工作，并指导你如何提交错误报告。你会通过编程实战学习到全新的技术知识，这些会对你未来的项目设计开发和自身的技术水平提升有很大的帮助。

我们将为HTML5和CSS3多个规范编写测试，这将有助于测试浏览器对这些标准的实现，并在它们表现不一致时提交错误报告。我们还将听到专家们对这些标准的意见以及浏览器如何实施这些标准的信息。
